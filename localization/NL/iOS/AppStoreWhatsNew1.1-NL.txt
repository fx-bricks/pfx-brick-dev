The PFx Brick App for iOS has many new features and enhancements as well as being available in serveral new languages.  These features include:
- improved Bluetooth device scanning and filtering
- edit Startup Actions directly from mobile App
- edit any remote control action without desktop app
- refresh audio file listing
- improved layout metrics for iPhone 8, iPhone X
- now available in German, Dutch, and Swedish
- miscellaneous bug fixes and reliability enhancements
